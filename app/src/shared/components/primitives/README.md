# Primitives

Unbranded atoms and thin MUI-aligned wrappers live here.

Use this slot for reusable component foundations such as buttons, text fields,
tooltips, headings, and icon-only controls when a project intentionally promotes
them beyond the existing flat `src/shared/components` wrappers.

Follow MUI's component split and prop vocabulary when there is a clear MUI
counterpart, then layer scaffold tokens or brand semantics in the caller or
brand wrapper.
